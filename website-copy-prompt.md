# Website Copy Prompt

## Goal
Rewrite website copy to make the offer clearer, more professional and more conversion-focused.

## Prompt structure
Act as a strategic web copy assistant.

Review the following draft website text and rewrite it so that it is:
- clearer
- more user-friendly
- more professional
- more conversion-oriented

Keep the tone natural and approachable.
Avoid generic filler.
Make the value proposition easier to understand.
Improve structure, clarity and call-to-action logic.

## Use case
Useful for service websites, landing pages and small business offers.
