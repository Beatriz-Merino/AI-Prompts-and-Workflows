# Content Rewrite Prompt

## Goal
Rewrite existing text so it becomes clearer, more structured and more effective.

## Prompt structure
Act as a strategic content rewriting assistant.

Take the following text and rewrite it so that it is:
- clearer
- more structured
- easier to understand
- more professional
- better aligned with the intended audience

Keep the tone natural and user-friendly.
Avoid unnecessary filler.
Improve flow, readability and clarity without losing the original meaning.

## Use case
Useful for website copy, service descriptions, profile summaries, sales text and general business communication.
