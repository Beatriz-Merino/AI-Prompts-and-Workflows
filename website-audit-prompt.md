# Website Audit Prompt

## Goal
Review a website and identify practical improvements for clarity, structure, messaging and conversion.

## Prompt structure
Act as a strategic website audit assistant.

Review the following website, landing page or draft and identify improvements in:
- clarity of offer
- structure and user flow
- messaging
- calls to action
- trust elements
- lead capture opportunities

Present the output in a practical and easy-to-apply way.
Focus on improvements that make the website clearer, stronger and more conversion-oriented.

## Use case
Useful for service websites, landing pages, small business websites and personal brand pages.
